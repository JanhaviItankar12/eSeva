# ESeva - District-Level E-Governance Platform



## 🌟 Project Overview
**ESeva** is a district-level e-governance platform aimed at digitizing certificate and document processing in government offices.  
It ensures **transparency, accountability, and faster citizen service** through multi-tier verification, automated notifications, and intelligent monitoring.

The system targets **citizens, government employees, officers, and senior district officials** for efficient workflow management.

---

## 🏛️ Features (Implemented & Ongoing)

### ✅ Completed Features (Admin + Citizen)
- Citizen registration & login
- Online application submission for certificates
- Document upload
- Application tracking by ID
- Status view: Pending / Approved / Rejected
- Automatic email notifications
- Admin dashboard: manage users and offices

### 🔄 Ongoing Features
- Multi-level verification workflow (Gram Panchayat → Tehsil → District)
- Clerk and Officer roles
- Correction requests and back-and-forth verification
- Tehsildar and District Officer dashboards
- ML-based delay prediction and abnormal office detection
- Analytics: pending ratio, rejection ratio, SLA compliance

---

## 👥 User Roles
1. **Citizen** – Apply for certificates, track status, receive notifications  
2. **Admin** – Manage users, offices, and monitor system  
3. **Clerk / Gram Sevak** – Verify documents at Gram/Tehsil level (upcoming)  
4. **Officer / Tehsildar** – Approve/reject applications, send for correction (upcoming)  
5. **District Officer** – Monitor all offices, ML insights, audits (upcoming)

## 💻 Tech Stack
- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js, JWT authentication  
- **Database:** MongoDB  
- **Notifications:** Nodemailer (Email)  
- **ML (Planned):** Python, Scikit-learn, Flask/FastAPI
